# Room WiFi Logger

## アプリ概要

Room WiFi Logger は、部屋ごとに Wi-Fi の電波強度を記録するための Android アプリです。

部屋を選んで測定を開始すると、接続中の Wi-Fi アクセスポイントの RSSI を約30秒間サンプリングし、平均・最小・最大 RSSI などを端末内に保存します。保存した記録は一覧で確認でき、CSV として端末の `Downloads` フォルダへ出力できます。

自宅や小規模な室内環境で「どの部屋の Wi-Fi が弱いか」を手元で記録し、あとから Excel などで比較したい人向けの個人用ツールです。

## 主な機能

- 部屋の追加、名前変更、削除
- 部屋ごとの Wi-Fi 電波強度測定
- 約30秒間、約3秒間隔で RSSI を自動サンプリング
- 平均 RSSI、最小 RSSI、最大 RSSI、サンプル数を保存
- SSID、BSSID、周波数帯、リンク速度、メモを測定記録に保存
- 測定履歴の一覧表示
- 部屋で絞り込んだ履歴確認
- 全測定データの CSV エクスポート
- UTF-8 BOM 付き CSV 出力で、日本語の部屋名を Excel で開きやすい形式に対応

## インストール

1. GitHub の [Releases](../../releases) から最新の APK をダウンロードします。
2. Android 端末で APK を開き、画面の案内に従ってインストールします。
3. 必要に応じて、端末設定で「不明なアプリのインストール」を許可します。
4. 初回測定時に位置情報の権限を許可します。

対応環境:

- Android 13 以降
- Wi-Fi 接続中の端末
- 位置情報サービスがオンになっていること

Android では接続中 Wi-Fi の SSID や BSSID を取得するために位置情報権限と位置情報サービスが必要です。このアプリは GPS 位置情報の記録や外部送信は行いません。

## 使い方

1. アプリを開き、測定したい部屋を追加します。
2. 部屋を選択します。
3. 必要ならメモを入力します。
4. 測定開始ボタンを押します。
5. 測定が完了するまで端末をその部屋に置くか、同じ場所で保持します。
6. 測定完了後、履歴画面で結果を確認します。
7. CSV が必要な場合は、履歴画面から CSV エクスポートを実行します。

CSV は `wifi_log_yyyy-MM-dd.csv` という名前で端末の `Downloads` フォルダに保存されます。
